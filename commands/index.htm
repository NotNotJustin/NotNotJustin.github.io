    • Channel & Bot Commands

    A list of the Channel and Bot Commands for NNJ's Twitch Channel


    ⤓




















    Channel Commands

    Location    Type    Command    Usage    Description    
    Twitch    Custom    age    !age    Tells you how long ago and on what date your Twitch account was created.    
    Twitch/Discord    Module    redeem    !redeem [PRIZE ID]    Allows for redemption of the prizes listed on https://notnotjustin.ml/prizes. Make sure to match the ID in the command with the one on the website.    
    Twitch    Module    bank    !bank    Base Command for the in-channel bank.    
    Twitch    Module    fc    !fc [@USERNAME]    Check the Nintendo Switch Friend Code of a user. Omit @username to post your own.    
    Twitch    Module    myfc    !myfc [Nintendo Switch FRIEND CODE]    Update the database so it includes your Nintendo Switch FC. Check it has posted with !fc .    
    Twitch    Custom Game    heist    !heist [BET IN NNJCoins]    Start a Bank Heist! Bets can range from 10 NNJCoins to 1000 NNJCoins! Be wary, though! If the cops catch you, you have to wait until the heat dies down before taking another crack at the vault. (Robbers must own a Tamagotchi. See Command below.)    
    Twitch    Custom Game (Dependency)    tamagotchi    !tamagotchi    Base command for the Tamagotchi Game. Dependency for Heist game.    
    Twitch    Custom Game    tgbuy    !tgbuy    Buy a Tamagotchi from the store. Must perform this command before doing Heists!    
    Twitch    Custom Game    tgfeed    !tgfeed    Feed your Tamagotchi so it may gain EXP! (Must spend NNJCoins to buy feed.)    
    Twitch    Custom Game    tgkill    !tgkill    You did what must be done.
    Twitch    Custom Game    steal    !steal [@USERNAME] [NNJCoin Amount]    Steal NNJCoins from a fellow viewer. Oh, they will be mad! Min. Amount: 1 NNJCoin; Max. Amount: 100 NNJCoins    
    Twitch    Default    Check Link in Description.    Check Link in Description.    Full list of Default Commands located at https://phantombot.tv/guides/#guide=content/commands/commands . Most of these commands can only be used by NNJ, a Twitch Moderator, or a Bot Manager.    
    Twitch    Custom    commandlist    !commandlist    Gives a link to this webpage.    
    Twitch    Custom    curator    !curator    Gives a link to the Steam Curator page on the NNJ Website.    
    Twitch    Custom    discord    !discord    Gives the invite link to the #NotNotNation Discord Server.    
    Twitch    Custom    followage    !followage [@USERNAME]    Tells how long another user has been following the channel. Omit @USERNAME to find out how long you have been following.    
    Twitch    Custom    game    !game    Displays information about the currently streamed game.    
    Twitch    Custom    modapp    !modapp    Tells the user the requirements for becoming a Twitch Moderator on NNJ's channel.    
    Twitch    Custom    patreon    !patreon    Tells the user the link to and some info about NNJ's Patreon.    
    Twitch    Custom    ping    !ping    Pong! A simple test command to see if the bot is online and running properly.    
    Twitch    Alias    bal    !bal    Alias for the !coins command. You may use either !bal or !coins to check your current NNJCoins Balance.    
    Twitch    Custom    playtime    !playtime    Tells the user how long the streamer has been playing the currently streamed game. Resets on Game Change or Crash.    
    Twitch    Custom    prizes    !prizes    Announces the link to the Prizes page on the NNJ Website.    
    Twitch    Custom    sens    !sens    Tells the user what sensitivity Justin usually plays Destiny 2 on. Applys to both M&K and Controller gameplay.    
    Twitch    Custom    sponsors    !sponsors    Tells the user Justin's Sponsor links and codes.    
    Twitch    Custom    tip    !tip    Tells the user how to tip the stream.    
    Twitch    Custom    title    !title    Displays the current stream title to the user.    
    Twitch    Custom    uptime    !uptime    Tells the current stream uptime, or how long the stream has been live.    
    Twitch    Custom    wds    !wds    Announces to the stream what web design software Justin uses.
    Twitch    Custom    website    !website    Gives a link to this website.    
    Discord    Default    Check Link in Description.    Check Link in Description.    Full list of the NNJBot Discord Bot's Default Commands available at https://phantombot.tv/guides/#guide=content/commands/discord-commands . Note that most commands are management commands, only for use by the Broadcaster, Discord Admins, or Bot Administrators.
